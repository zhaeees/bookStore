# 📚 북스토어 (Bookstore)

간단한 온라인 서점 웹사이트입니다.
React + Vite 기반으로, 실제 서비스처럼 보이는 **레이아웃 / 반응형 / UI 흐름**을 중심으로 구현했습니다.

> 💡 학습용 프로젝트로, 구조 설계와 의사결정 과정을 함께 정리했습니다.

---

## 🧩 1. 프로젝트 개요

* **프로젝트명**: 북스토어 (Bookstore)
* **한 줄 정의**:
  👉 더미 도서 데이터를 기반으로 **홈 / 검색 / 상세 / 마이페이지 흐름을 구현한 리액트 서점 UI 프로젝트**

---

## 🛠️ 2. 기술 스택

| 구분         | 기술                                              |
| ---------- | ----------------------------------------------- |
| Framework  | React ^18.x                                     |
| Build Tool | Vite ^5.x                                       |
| Routing    | React Router ^6.x                               |
| UI         | react-icons, swiper                             |
| Styling    | 순수 CSS (reset / variables / global / component) |

> 📌 *버전은 package.json 기준*

---

## 🚀 3. 실행 방법

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev
```

✔ 실행 후 브라우저에서 확인 가능

## ✨ 5. 핵심 기능

### 🏠 홈 화면

* Swiper 기반 메인 비주얼 슬라이더
* 베스트 / 신간 / 특가 / 추천 도서 섹션 구성

### 🔍 검색 & 필터

* URL Query 기반 필터링
* 카테고리 + 정렬 (최신순 / 평점순 / 가격순)

### 📖 도서 상세

* `:id` 기반 단일 도서 조회
* 가격 / 할인 / 리뷰 / 장바구니 기능

### 👤 마이페이지

* 탭 UI 구조
* 장바구니 / 찜 / 주문 / 내정보

---

## 🧠 6. 주요 설계 포인트

### 📌 레이아웃 설계

* `MainLayout`으로 공통 구조 통일
* `.inner (1200px)` 기반 정렬 시스템

### 📌 상태 관리

* 장바구니만 `Context`로 전역 관리
* 나머지는 로컬 상태(useState)

### 📌 데이터 구조

* JSON 기반 더미 데이터
* → 추후 API로 쉽게 교체 가능하도록 설계

---

## 📱 7. 반응형 전략

* 모바일 헤더 재배치 (order 활용)
* 필터 영역 `flex-wrap` 처리
* 슬라이더는 full width + 내부만 제한

---

## ⚠️ 8. 트러블슈팅

### ❌ 문제 1: 모바일 가로 스크롤 발생

👉 원인: 필터 영역이 한 줄 고정

✔ 해결:

* `flex-wrap` 적용
* 자연스럽게 줄바꿈 처리

---

### ❌ 문제 2: 모바일 헤더 구조 붕괴

👉 원인: 요소 우선순위 없음

✔ 해결:

* `order`로 재배치
* 햄버거 메뉴 구조 도입
